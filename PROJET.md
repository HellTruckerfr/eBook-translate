# eBook Translate

Outil de traduction d'EPUBs anglais vers le français.

## Dépendances
- Mistral Large API (traduction)
- ebooklib (lecture/écriture EPUB)
- FastAPI (backend)
- React + Tailwind (GUI)

## Sorties
- EPUBs français (un par arc)
- Fichiers TXT (un par chapitre)
- Fichiers JSON (pipeline audiobook)
